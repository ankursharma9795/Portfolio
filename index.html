Tic Tac Toe

Tic Tac Toe is a classic two-player strategy game played on a 3x3 grid. Players take turns marking X or O in empty squares. The first player to align three of their marks horizontally, vertically, or diagonally wins. If all squares are filled without a winning line, the game ends in a draw. It's simple, fun, and helps improve logical thinking.


-> HTML: Structures the game board using elements like "div" and "table"

-> CSS: Styles the board, cells, player marks (X/O), and adds hover or transition effects.

-> JavaScript: Handles game logic such as turns, win conditions, draws, and resetting the game.

LoginPage

A login window is a user interface form that allows users to securely enter their credentials. It typically includes input fields for email or username and password. HTML is used to structure the form elements, while CSS styles the layout, spacing, and colors. The form often includes a "Remember Me" checkbox and a login button. CSS helps center the form, add hover effects, and make it responsive across devices. This basic login page can be integrated with authentication systems using JavaScript or back-end code.
